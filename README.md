## 📋 Описание проекта

**Random Task Generator** — это графическое приложение для генерации случайных задач, которое помогает организовать повседневные дела, добавляя элемент случайности и ведения статистики.

### Основные возможности

- ✅ **Генерация случайных задач** из предопределённого списка
- ✅ **Фильтрация** по трём типам: учёба, спорт, работа
- ✅ **Добавление пользовательских задач** с валидацией
- ✅ **Автосохранение истории** в JSON файл
- ✅ **Загрузка истории** при запуске приложения
- ✅ **Статистика** выполненных задач с графиками
- ✅ **Валидация ввода** (пустые строки, длина, дубликаты)
- ✅ **Очистка истории** с подтверждением
- ✅ **Визуальная обратная связь** (мигание кнопки, цветовые индикаторы)
