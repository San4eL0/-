# Random Task Generator

**Автор:** Александр Шило  
**GitHub:** https://github.com/San4eL0/-.git
## Описание

Графическое приложение на Python (Tkinter) для генерации случайных задач по категориям.  
Функции:
- Генерация случайной задачи из списка
- Добавление новых задач с выбором типа (учёба, спорт, работа)
- Фильтрация истории по типу задачи
- Сохранение всех задач и истории в JSON
- Валидация ввода (пустые строки не допускаются)
