## 📝 Описание проекта

Random Task Generator — это графическое приложение на Python с интерфейсом Tkinter, которое помогает генерировать случайные задачи по категориям (учёба, спорт, работа). Программа сохраняет историю сгенерированных задач и позволяет добавлять новые задачи.

### Основные возможности:

- ✅ Генерация случайной задачи из списка
- ✅ Добавление новых задач с выбором типа
- ✅ Фильтрация истории по типу задачи
- ✅ Сохранение задач и истории в JSON файлы
- ✅ Валидация ввода (проверка на пустые строки)
- ✅ Очистка истории
- ✅ Красивый и удобный интерфейс
